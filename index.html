AppBuilder Workflow CoT

不仅仅是问答，更是你的飞行管家。

AutoFlight 是一款基于百度千帆 AppBuilder 开发的 AI 原生应用。它能理解复杂的飞行需求，识别“红眼航班”、“宽体机”等行业术语，并结合 API 熔断机制提供高可用的智能决策。

  • 思维链 (CoT)：先需求挖掘，再风控分析，后给出建议。
  • 容错机制：API 异常时自动降级，保证服务不中断。
  • 专业排版：结构化输出，卡片式展示，信息一目了然。

架构迭代：从工具流到 Code-Based

初期尝试使用平台原生的 GUI 工具流搭建，但由于平台对外部 API 接口的配置不够稳定，导致流程经常中断。

❌ V1.0 纯工具流

拖拽式连线，API 调用不稳定，调试困难。

✅ V2.0 自然语言编排

利用 Prompt 编排逻辑，配合插件调用，灵活性与稳定性大幅提升。

搭建心得 & 关键技术点

RAG 知识库

输入了四个专业文件构建知识库，有效避免模型幻觉。

思维链 (CoT)

提示词策略：先进行“需求挖掘”和“风控分析”，再给出行动建议。

行业黑话识别

识别“红眼航班”、“宽体机”、“同城异地机场”等术语，打造专业顾问人设。

API 熔断机制

若接口配置未生效，自动降级为“经验模式”给出通用建议，保证演示不翻车。

结构化排版

强制使用 Markdown 表格和 Emoji，提升信息可读性。

多插件协同

集成了智能搜索生成、网页总结、实时航班查询、天气查询等多个工具。
